# Customer Retail Machine Learning Project

## Project Overview

This project compares multiple Machine Learning algorithms using a customer retail dataset. The objective is to preprocess data, train different classification models, evaluate their performance, and compare results.

## Dataset Features Used

* Quantity
* UnitPrice
* Country

## Machine Learning Models

1. Logistic Regression
2. Decision Tree Classifier
3. K-Nearest Neighbors (KNN)

## Project Workflow

* Data Loading
* Missing Value Handling
* Label Encoding
* Data Visualization
* Train-Test Split
* Model Training
* Accuracy Evaluation
* Confusion Matrix Analysis
* Model Comparison

## Evaluation Metrics

* Accuracy Score
* Confusion Matrix

## Visualizations

* Customer Distribution Graph
* Model Accuracy Comparison Graph

## Conclusion

This project demonstrates the complete Machine Learning workflow. Multiple algorithms were trained and evaluated on the same dataset. Their performance was compared using accuracy scores and confusion matrices to identify the most suitable model.
